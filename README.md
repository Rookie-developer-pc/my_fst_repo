## 🛠️ 技术详情 (Tech Details)

* **开发环境**：Android Studio Ladybug
* **核心语言**：Kotlin
* **布局技术**：XML / ViewBinding (计划升级至 Jetpack Compose)
* **架构模式**：正在由基础模式向 **MVVM** 架构进化
* **数据存储**：目前为内存存储 (Next Step: Room Database)
